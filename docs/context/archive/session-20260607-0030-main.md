## Key Decisions

### 1. ADR-0027 — Figure block shape and serialization

### 2. ADR-0028 — Caption editability resolved to Option A
Option A (editable inline caption via `setFigureCaption` reducer arm + `EditorLayout` caption segment) chosen to honor ADR-0023's "real, editable block" promise. Reducer arm and editable segment are LT4-2 deliverables.

### 3. ADR-0009 amendment — `figure` added to closed vocabulary
`figure` joins the closed block vocabulary; its reveal representation is the `[figure: <ref>]` chip; actual image rendering and placement remain deferred to the typesetter per ADR-0024.
