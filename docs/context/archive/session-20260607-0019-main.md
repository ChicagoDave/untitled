## Key Decisions

### 1. ADR-0026 — Atomic reducer arm for section inserts; closed `SectionRole`
Decision (a): `SectionRole` is a closed enum (`chapter|prologue|epilogue|dedication`) on `ChapterCut`, serialized as its own inverse `String` raw value. A free-form title string is insufficient for typesetter-driven layout (ADR-0024 contract).
Decision (b): Single atomic `InputEvent.insertSection` reducer arm chosen over composing `insertBlock` + `placeChapterCut` — the minted `BlockID` for the seeded paragraph is only available inside the reducer; composing two events opens a window in which the cut could be placed on the wrong block.
Decision (c): Titling has exactly one home — the inline edit on the reveal chip / main-editor heading. A palette-time prompt was built and removed the same session.
Decision (d): Titles are never empty; inserting a section seeds the role's `defaultTitle`; clearing a title reverts to the default. Numbering uses `#a`/`#r` macros resolved role-aware at render time (stored title preserves the macro per ADR-0024).
