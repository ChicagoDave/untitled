## Key Decisions

### 1. ADR-0017: Shell-script bundle assembly
`scripts/bundle.sh` on the `swift build` spine (not Xcode, not third-party plugin) preserves the headless CI path and keeps assembly transparent. Info.plist lives at `app/Packaging/Info.plist` (not inside `Sources/`) to avoid an unhandled-resource error from SwiftPM — a concrete deviation from the session plan, documented in the ADR and plan.md Status field.

### 2. ADR-0018: AppDelegate open-document routing; DocumentGroup rejected
`WindowGroup` + `@NSApplicationDelegateAdaptor AppDelegate` is additive (one delegate method) and preserves ADR-0011's separation. `DocumentGroup` would have required restructuring the entire app around `FileDocument`/`ReferenceFileDocument`. `AppWorkspace.shared` is the bridge between the delegate's early Apple-event context and the SwiftUI scene.

### 3. ADR-0019: .galley as macOS package type (complements ADR-0007)
Package type is a Finder/OS perception layer only. `prose.txt` stays plain text and independently openable in any text editor. Switching to a zip or binary container is explicitly ruled out. ADR-0019 is a complement to ADR-0007, not a supersession.
